#### How can I calculate the sum of discounted books?
#### you need call the function `calculateDiscountedBooks` and pass an array of strings with the books you want to buy.

```
const books = ['Harry potter le grand maître', 'Harry potter la chambre des secrets', 'Harry potter le prisonnier d\'Azkaban']

calculateDiscountedBooks(books)
```
